# 3dmigoto 加载器下载

不同的游戏需要使用其专用的 3dmigoto 版本才能正常加载 mod。你可以在这里找到部分游戏 3dmigoto 的管理器适配版压缩包，然后参考 [添加加载器包](/help/tutorial-loader.html#添加加载器包) 添加进管理器进行使用；也可以跳转对应游戏专用 3dmigoto 的官方下载链接获取。
