# modify_object_name
编辑对象名称插件

## 功能简介

该插件可以使对应对象下的所有模组的作用对象修改为新的对象名称

## 使用教程

### 加载插件
首先，也是最基础的一步——**加载插件**，请参考 [插件使用教程](/help/tutorial-plugins)

### 编辑对象名称
鼠标移动至要编辑的作用对象上，鼠标右键呼出菜单，选择 **修改对象名称**，在打开的编辑窗口修改对象名称。修改完成后，点击 `保存` 按钮即可使该作用对象下的所有模组的作用对象修改为新的对象名称。
