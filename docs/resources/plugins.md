# 扩展插件

插件是一个实验性质的功能，它允许你加载额外的代码和资源，以丰富和扩展程序功能。
具体的使用方式请参考 [插件使用教程](/help/tutorial-plugins) 。

<weaken>使用 Ctrl + F 在页面内快速搜索</weaken>
